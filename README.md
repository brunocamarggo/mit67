# Prerequisites
* [OpenCV](https://opencv.org/)
* [Opscikit-learn](https://opencv.org/)

# Installing
Just set the dataset paths in `config.py` file.
```
# -*- coding: utf-8 -*-
paths = dict(
    ANNOTATIONS='/example/of/path/MIT67/Annotations/',
    IMAGES='/example/of/path/MIT67/Images/'
)

```
# Running
```
$ python mit67.py
```
